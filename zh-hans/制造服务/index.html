制造服务

我们高度专业化的团队能够为客户制造广泛系列的各种产品。我们的专业服务还包括苛刻应用所需的原型产品、装配组件以及小型与中型配件的工程、设计与制造。此外，我们还深受客户信赖，不仅负责管理所有物流，还负责在产品的整个生命周期中提供产品支持服务。 我们涉足广泛的行业和应用，可为（但不仅限于）下列行业提供高度专业化的服务：航空航天、牵引、自动化、能源、机床、医疗技术、包装及印刷、总装建造和康复技术。

金属板材加工

作为客户最信赖的伙伴，我们积极与客户展开密切合作，提供尖端的金属板材加工制造解决方案。我们的专家不仅能够缩短板材制造的周期时间，还能设计及随时制造出创新、低成本、高质量的部件和装配组件。我们完整系列的工具和技术将确保我们能够迎接任何金属加工方面的挑战，同时实现功能性和生产率的完美结合。 我们的金属加工能力包括： 工程：三维CAD /二维CAD / 三维编程 / 二维编程 反向工程：我们的扫描技术能够以激光的速度捕捉现有部件或模板的完整形貌并将数据保存在兼容CAD的文件中，以备将来之用。 金属板材加工：激光切割 / CNC冲压 / 激光冲压 / 外心压力机和水压机和冲压机 成型：CNC刨边/车圆/压纹/深冲压 连接：铆接/咬接/焊接/锻接/点焊 后处理：研磨/刷光/电镀/喷涂 装配：随时可进行安装的产品/ 及时交付的项目

高价值零部件制造

我们能够设计和制造从简单到高精度的设备部件，包括使用难以加工的材料制造的部件。我们的专家可从初始原型、功能测试到制造方面，提供各种必要的指导。我们广泛的制造选项（铣削中心、车床、表面和外圆磨床、电火花线切割机和立式放电加工机以及各种夹具镗孔机）能够让我们针对各种应用提供最佳的制造技术。 我们的高价值零部件制造范围包括： 工程:三维CAD /二维CAD / 三维编程 / 二维编程 我们的开发和设计团队能为客户提供用于重新设计或优化产品­、模块、 夹具和固定装置或工具的解决方案。 加工的构件：我们具备多台先进的机床，可采用广泛的制造技术，例如铣削、钻孔、车削、研磨和EDM配线，加工尺寸最大为2000 x 1200 x 1500毫米。 夹具、固定装置和工具:我们擅于设计和制造生产设备、夹具和固定装置、量具和测试设备­，并能够提供非常精密的夹紧设备、装配设备和工件夹具。 装配制造:我们能通过在构建和装配随时可用的系统和随时可安装的模块时对物流和采购实施管理，从而精简许多制造环节，进而提高客户的满意度。 材料：我们专门加工各种材料，包括铝铜合金、钛镍基合金（例如Hastelloy X、Inconel、Nimonic 和 Haynes）。

磨刀服务

我们完全了解，通过优化刀口的几何结构，往往便能提高切割工具的有效性和一般使用寿命。因此，我们的专家均能帮助客户选择对工具进行磨刀或喷涂的正确原材料，而我们一流的经验通过采用创新的磨刀技术，也能为客户提高潜在的效率。 我们的磨刀范围包括： 常规磨刀：全方位系列或服务 CNC磨刀:标准/圆柱形/圆锥形/定制工具

热处理

我们的热处理服务采用传统工艺，例如在正常环境、惰性气体环境或真空环境下进行退火、硬化、钎焊和回火。另外，我们还提供各种特殊工艺（例如渗氮/氮碳共渗工艺，这是一种环保的表面硬化工艺）以及所需的前期和后期处理工艺。此外，通过我们的专业知识和基础设施，我们还能提供独一无二的铝合金淬火和回火服务以及钢和铜材料热处理。 我们的热处理范围包括： 真空系统：我们的高真空炉适合进行退火、硬化和钎焊。最大批次尺寸为1200 x 900 x 1500毫米，淬火硬化能力从四个面一直到五条筋。 钎焊：我们拥有配备特殊设备的部件准备室，以及点焊设备和自动焊料分配器，并专门使用镍基爆料进行钎焊。 回火：我们的铝部件回火系统利用加热水槽，可对长800毫米、直径为1200毫米的部件进行回火。 硬化、退火：我们备有用于淬火、碳化、硬化、退火、固溶处理和沉淀硬化的热油槽、氮和氩。 后处理：我们使用压力喷丸系统（喷射沙石或玻璃球）对部件进行抛光。我们还提供机械式前处理和后处理服务。 气体渗氮和氮碳共渗：这些表面处理工艺能够使机械设备格外耐磨，对于尺寸高达1500 x 1000毫米的齿轮组件、齿轮、机床和模具部件来说尤其如此。我们的渗氮传感器可监督和控制整个表面过程，以确保达到所要求的质量，其中包括：实现较高的表面硬度、最大的耐磨性、更高的耐腐蚀性、更佳的滑动特性，以及最小的尺寸变化或变形。

工具制造和租赁服务

新工具：我们能够按照具体的要求制造新工具。当中，我们能制造铣刀、钻头、CNC中心销和其他各种工具。我们的专家还可以帮助客户设计或完善工具的规格，确保材料获得最高效的利用和提供最佳的产品质量。   租赁测量仪器、标准工具和定制工具。在许多情况下，根据需要来租赁设备而不是购买和维护不太需要的设备是一种更具成本效益的做法，而对于特大型工具而言尤其如此。我们的租赁服务可在一系列具有吸引力的条款和条件下，提供广泛的租赁选项。

产品举例

燃气和蒸汽轮机部件：

  • 燃气轮机燃烧室、热气通路部件、燃气导流板
  • 加工叶片或桨叶、叶片装配部件（例如，闭锁装置和垫片）、扩压器、定子段和隔热装置、蒸汽和燃烧器过滤器
  • 翻新燃气轮机叶片和隔热装置（拆除和重新焊接蜂窝密封件，包括提供质量检验）
  • 水力和涡轮发电机部件：
  • 完全在真空环境下成形、加工和焊接的端子和导电棒
  • 制造电机定子铁芯组件，包括相关冲压工具的工程和制造。
  • 外部支持和辅助系统：
  • 齿轮泵、阀、致动器和压力变送器（用于蒸汽和润滑油供应）
  • 转子旋转装置和其他齿轮部件
  • 工厂和工地设备
  • 特种设备的工程、制造和调试
  • 定制测量和检测工具
  • 用于涡轮机和发电机部件的装配设备、运输和吊装设备
  • 配备齐全的工具柜（配备有符合客户规格的特殊工具）
  • 用于在转子上进行铣削和钻孔操作的特殊设备
  • 用于加工导电棒的特殊设备（冲孔/切割、折弯和测量）
  • 用于轮机部件压力测试和流速测量的测试设备。
  • 设计和制造研磨、铣削、EDM和装配夹具（用于制造轮机叶片）。
